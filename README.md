# Padel Opponent Finder

A React + Vite app for discovering padel courts and finding players looking for matches nearby.

## Features

- Interactive court map (Leaflet) with player-count markers
- List view with searchable courts and booking links
- Player modal for match discovery at each court
- Mobile-friendly layout:
  - Dynamic viewport height (`dvh`) handling
  - Safe-area support for iOS notches/home indicator
  - Larger touch targets for key actions
  - Improved small-screen button/layout stacking

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Lucide icons
- Leaflet (loaded at runtime)
